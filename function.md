## 函数

### 数学中的函数
传入同样的`x`一定会得到同样的`y`

`f(x) = y`

我们平常用到的是编程里的子函数

### 函数中的返回值由什么决定？
影响因素：  
* 调用时输入的参数`params`
* 定义时环境的`env`

我们看下面的一个例子：  
```js
let x = 'x';
let a = 1;

function f1 (x) {
  return x + a;
}

{
  let x = 'x';
  let a = 2;
  console.log(f1(x)); // "x1"
}
```

这里的`a`我们用到的是定义时的`a`,而不是执行时的`a`。
